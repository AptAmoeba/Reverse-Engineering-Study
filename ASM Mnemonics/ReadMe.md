## Recognizing Assembly Instructions In Hexadecimal
Seeing a lot of lone 4x bytes in hex are likely MOV instructions, and if you see a lot of 4x5x bytes bunched together, it's likely real, unencrypted data/code. 

&nbsp;

## Recognizing Consistent Routines in Assembly 
If you see gs: and 60 in an assembly snippet, the malware is likely PEB Walking. (source: m4hedgehogs).
