## Making Different View Windows
<img width="907" height="226" alt="image" src="https://github.com/user-attachments/assets/4fca28d0-1cfb-4077-80dc-1455281115e2" />

Use the "Split View" option (shown in top-right) to make a new Pane, and you can select what kind of view you want it to be at the top.

My standard is (Left) High Level IL, (Right) Disassembly

If you click-and-hold the top of any view pane, you can rearrange it within Binja, similar to the Windows Snap pane feature.

## Linking Windows to Sync with mouse selections

Click the link icon and set them both to the same Sync Group.

<img width="189" height="101" alt="image" src="https://github.com/user-attachments/assets/86706abb-8549-4d66-8039-51b6418df8ca" />

Now, with them both on the same Sync Group, any click/highlight in one pane will go to the same location in the other!
